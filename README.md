<h1>Winter Memory Game</h1>
<p>
A winter-themed memory game where you have to find 8 pairs in the shortest time and with the fewest moves. <br>
The game is made with HTML, CSS and JavaScript, with a background of aurora borealis and falling snowflakes. <br>
If you find the pair within 1 second, you get a bonus move that does not count towards to your attempts. <br>
Press the "New Game" button to start and try to find all pairs as quickly as possible! <br>
</p>
